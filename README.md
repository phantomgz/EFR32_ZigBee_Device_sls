# EFR32_ZigBee_Base_Proj_sls

#### Description
一些常用的zigbee设备基本工程，以sls文件上传，导入后更改chip即可作为产品最初的版本使用。
一般是基于Z3lightSoc, 主要修改了ZCL.


工程列表
1. 零火线开关（Router)
2. 单火线开关 (enddevice sleepy)
3. 网关（coord)
